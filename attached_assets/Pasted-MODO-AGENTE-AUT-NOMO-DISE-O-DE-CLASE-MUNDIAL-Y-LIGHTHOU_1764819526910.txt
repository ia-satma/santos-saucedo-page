PARTE 1: TENDENCIAS DE DISEÑO 2024-2025 A IMPLEMENTAR
═══════════════════════════════════════════════════════════════

1. BENTO GRID LAYOUTS
   - Implementar layouts estilo Bento Box en secciones de servicios
   - Grids asimétricos pero balanceados
   - Cards de diferentes tamaños que crean jerarquía visual
   - Ejemplo: áreas de práctica en grid bento, no grid uniforme aburrido

2. GLASSMORPHISM SUTIL (NO EXCESIVO)
   - Usar backdrop-filter: blur() en elementos flotantes
   - Cards con fondo semi-transparente sobre imágenes
   - Navegación sticky con efecto glass
   - Opacidad controlada: no más de 0.8
   - Solo donde aporte, no en todo el sitio

3. GRADIENTES MESH Y AURORA
   - Fondos con gradientes suaves estilo aurora
   - Usar para secciones hero o separadores
   - Colores de marca Santos & Saucedo: tonos azul marino, dorado, blanco
   - Gradientes sutiles, profesionales, no neón

4. TIPOGRAFÍA VARIABLE Y CINÉTICA
   - Implementar variable fonts para mejor rendimiento
   - Títulos con peso dinámico en hover
   - Micro-animaciones en texto hero
   - Kerning y tracking refinados
   - Escala tipográfica armónica (ratio 1.25 o 1.333)

5. DARK MODE ELEGANTE
   - Implementar toggle dark/light mode
   - Dark mode con tonos profundos, no negro puro (#0a0a0a mejor que #000)
   - Mantener contraste WCAG AAA
   - Guardar preferencia en localStorage
   - Respetar prefers-color-scheme del sistema

6. MICRO-INTERACCIONES
   - Hover effects sutiles en cards (lift + shadow)
   - Botones con feedback táctil (scale 0.98 on press)
   - Links con underline animado
   - Iconos que reaccionan al hover
   - Transiciones de 200-300ms, easing cubic-bezier

7. SCROLL-DRIVEN ANIMATIONS
   - Elementos que aparecen al hacer scroll (fade up)
   - Parallax sutil en imágenes hero (no excesivo)
   - Progress bar de lectura en artículos
   - Navbar que cambia al scroll
   - Usar Intersection Observer, no scroll events

8. CURSOR PERSONALIZADO
   - Cursor custom para desktop
   - Efecto de hover que sigue el cursor
   - Cursor que cambia sobre elementos interactivos
   - Desactivar en móvil

9. NOISE TEXTURE Y GRAIN
   - Overlay sutil de grain en fondos sólidos
   - Agrega profundidad y sofisticación
   - Opacidad muy baja (0.03-0.05)
   - Usar SVG filter o imagen optimizada

10. ESPACIADO GENEROSO
    - Whitespace abundante entre secciones
    - Padding interno generoso en cards
    - Respiración visual entre elementos
    - Diseño que no se siente apretado

META: 100/100 en Performance, Accessibility, Best Practices, SEO

PERFORMANCE (100/100):

1. Largest Contentful Paint < 1.2s
   - Hero image en WebP, preloaded
   - Critical CSS inline
   - Font preload
   - Server response < 200ms

2. Total Blocking Time < 150ms
   - JavaScript defer/async
   - Code splitting
   - No long tasks (>50ms)
   - Web Workers para tareas pesadas

3. Cumulative Layout Shift < 0.1
   - Dimensiones explícitas en imágenes
   - Font-display: optional o swap con fallback similar
   - Reservar espacio para contenido dinámico
   - No insertar contenido arriba de existente

4. First Contentful Paint < 1.0s
   - Critical rendering path optimizado
   - Inline critical CSS
   - Preconnect a dominios externos
   - HTML mínimo initial

5. Speed Index < 2.0s
   - Progressive rendering
   - Above-the-fold priorizado
   - Lazy load below-the-fold

Implementar:
```html
<!-- En <head> -->
<link rel="preconnect" href="https://fonts.googleapis.com">
<link rel="preconnect" href="https://fonts.gstatic.com" crossorigin>

```

SCROLL ANIMATIONS:
```javascript
const observerOptions = {
  threshold: 0.1,
  rootMargin: '0px 0px -50px 0px'
};

const observer = new IntersectionObserver((entries) => {
  entries.forEach(entry => {
    if (entry.isIntersecting) {
      entry.target.classList.add('animate-in');
      observer.unobserve(entry.target);
    }
  });
}, observerOptions);

document.querySelectorAll('[data-animate]').forEach(el => {
  observer.observe(el);
});
```
```css
[data-animate] {
  opacity: 0;
  transform: translateY(30px);
  transition: opacity 0.6s ease, transform 0.6s ease;
}

[data-animate].animate-in {
  opacity: 1;
  transform: translateY(0);
}

[data-animate="fade-left"] {
  transform: translateX(-30px);
}

[data-animate="fade-right"] {
  transform: translateX(30px);
}

[data-animate="scale"] {
  transform: scale(0.95);
}

/* Respetar preferencia de movimiento reducido */
@media (prefers-reduced-motion: reduce) {
  [data-animate] {
    opacity: 1;
    transform: none;
    transition: none;
  }
}
```
